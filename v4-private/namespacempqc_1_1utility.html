◆ is_generator_v

template<typename T , typename Enabler = void>
constexpr bool mpqc::utility::is_generator_v = false
constexpr

Generator is a type that matches this concept:

template <typename T> concept Generator {
using value_type = T;
explicit operator bool() const;
Generator& operator++(); // only usable if this is true
const T& operator*() const; // only usable if this is true
};

and is comparable. Hence, Generator<T> is similar to a (constant) forward iterator.
