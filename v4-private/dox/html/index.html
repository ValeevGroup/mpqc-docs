Synopsis

The Massively Parallel Quantum Chemistry (MPQC) platform is a research package for ab initio simulation of the electronic structure of molecules and periodic solids. The current (4th) version of the package, internally referred to as MPQC4, is a modern revision of the conceptual design of the original MPQC platform using the massively-parallel tensor framework TiledArray, distributed task-based programming model and runtime MADWorld, and the Gaussian integrals library Libint.
