Detailed Description

Variants of canonical_column_phase() that use provided degenerate panel sizes.

Dispatches to canonical_column_phase_impl().

Parameters
[in]AThe matrix to be canonicalized
[in]A_element_comparison_tolerancethe soft comparison threshold used (if needed) by compute_pd_element_rowidx() to deduce the phase-defining elements; if negative, will use the default threshold used by canonical_column_phase_impl()
[in]reference_degeneraciesvector of degeneracies to use; the default is to assume each column is nondegenerate
[in]canonicalize_panel_orderwhether to reorder degenerate panels into canonical order (see permutation_to_canonical_column_order())
Returns
matrix U that canonicalizes the phase when A is right-multiplied by it (i.e. A.U)
