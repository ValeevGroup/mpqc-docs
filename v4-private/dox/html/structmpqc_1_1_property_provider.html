Detailed Description

utility class to support runtime casting to property provider

Every Property::Provider class derives from this. When possible every concrete property computer (e.g. wfn) will derive (via Provides<Properties...>) from this class and the default implementation of the provider method suffices. There are situations (e.g., Python) where the property computer cannot derive from Property::Provider classes and the cast must be performed via a table. In this case the virtual provider() method should be overridden accordingly.
