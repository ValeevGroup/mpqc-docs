Canonicalizes column phases and accounts for degeneracies among columns.

A column has canonical phase if the phase-defining element (see pd_element_rowidx ) in each column is a positive real number. This is an improved version of canonicalize_column_phase_nondeg() that can handle degeneracies among subsets of columns that allow these columns mix arbitrarily; to remove the arbitrariness of vectors in degenerate panels they are mixed unitarily to maximize the 4-norm of the vectors. Furthermore, the order of columns within degenerate panels and, optionally, the order of panels themselves is canonicalized (see permutation_to_canonical_column_order() for the definition of canonical order of columns/panels).

Note
in the context of LCAO canonicalization this function should receive not C but S.C
