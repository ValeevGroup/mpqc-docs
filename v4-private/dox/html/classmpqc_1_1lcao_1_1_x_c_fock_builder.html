Detailed Description

template<typename Tile, typename Policy>
class mpqc::lcao::XCFockBuilder< Tile, Policy >

XCFockBuilder computes the exchange-correlation energy and potential using GauXC integrator

mpqc::lcao::XCFockBuilder< Tile, Policy >::XCFockBuilder ( const KeyVal &  kv)

KeyVal constructor of XCFockBuilder

Parameters
kvthe KeyVal object; it will be queried for the following keywords:
Keyword Type Default Description
`functional` string none The functional to use for KS-DFT
`snK` bool false If true, will use seminumerical exact exchange builder implemented in GauXC
`grid` string fine The grid size to use; valid values are fine (75 radial points x 302 angular points), ultrafine (99 x 590), superfine (H, He: 175 x 974; rest: 250 x 974)
`kernels` array of strings none The list of kernels to use for KS-DFT
`weights` array of reals array of 1s The list of weights to use with the kernels for KS-DFT
`exec_space` string auto Controls where to execute GauXC; valid values are host, device and auto (to execute on device, if available, else on host)
`gauxc_device_fill_fraction` real 0.9 The device memory fill fraction for GauXC GPU execution; valid values are in (0,1)
`prune` string "unpruned" Pruning scheme for angular quadratures; valid values are unpruned (no pruning), robust (non-aggressive pruning), and treutler (aggressive pruning)
'batch_sz` integer 512 Target batch size for quadrature partitioning in the numerical integration
`snk_etol` real 1e-10 snK energy tolerance
`snk_ktol` real 1e-10 snK potential tolerance
`load_balancer_kernel` string Default GauXC execution kernel for the load balancer. See GauXC documentation for valid values.

Functional Composition Comment

SVWN5 SlaterExchange + VWN5
BLYP B88 + LYP

PBE0 0.75 PBE_X + PBE_C + 0.25 HFX
HF HFX Only useful with snK set to true
