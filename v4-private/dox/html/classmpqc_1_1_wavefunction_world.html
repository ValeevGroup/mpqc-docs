Detailed Description

It provides an execution context (madness::World) and a set of atoms.

WavefunctionWorld is an environment for one or more collaborating Wavefunction objects.

Parameters
kvThe KeyVal object; it will be queried for the following keywords:
Keyword Type Default Description
atoms Molecule or UnitCell none
molecule Molecule none This will be queried only if atoms is not given. This keyword is deprecated and may be removed in the future
speed_of_light_au double provided by default UnitFactory Turns on nonrelativistic limit for relativistic methods. This keyword takes precedence over fine_structure_constant since the speed of light is exact. The speed of light should be given in a.u.
fine_structure_constant double provided by default UnitFactory Turns on nonrelativistic limit for relativistic methods, ignored if speed_of_light_au is specified. This keyword is given a lower bound (~1e-8) due to the gap between electronic and positronic states increasing to infinity as the nonrelativistic limit is reached.
h1e string schrodinger whether to use non-relativistic Schrodinger 1-electron Hamiltonian or relativistic variants of the 1-electron Dirac Hamiltonian; valid values are `schrodinger` to use non-relativistic/Schrodinger Hamiltonian, `dirac` to use 4-component Dirac Hamiltonian (only works for MRA), `sf` to use spin-free/1-components X2C, `x2c` to use 2-component X2C, `x2c_sap` to use 2-component X2C constructed with SAP (superposition of atomic potentials), and `sfx2c_sap` to use spinfree/1-component version of the X2C constructed with SAP
electric_field std::vector<real> null if given, Hamiltonian will include the external field term. {x,y,z} components of the field are given in the vector in atomic units
multipole_origin std::vector<real> null {x, y, z} coordinates of multipole origin in the same units as the geometry input, if given, multipole expansion will be centered at this point
"$:world" pointer the value returned by mpqc::get_default_world() the default madness::World object to use

bool mpqc::WavefunctionWorld::spin_collinear ( ) const
Returns
true if Hamiltonian preserves spin collinearity (i.e. commutes with Sz),
Note
although this is property of the Hamiltonian, this determines the types of single-particle states; if returns true, single-particle (SP) states will be eigenfunctions of Sz, i.e. either spin-up or spin-down spin-orbitals or spin-free orbitals; if this returns false the SP states will be spinors/bispinors.
