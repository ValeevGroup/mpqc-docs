Detailed Description

template<typename Tile, typename Policy>
class mpqc::lcao::ExactKDiagonalBuilder< Tile, Policy >

ExactKDiagonalBuilder is an integral-direct implementation of FockBuilder in a Gaussian AO basis that uses 4-center integrals to compute the exact K diagonal and optimally takes / advantage of the permutational symmetry and shell-level screening.

similar to FockBuild::operator() , but computes the J+K part of the Fock matrix with variable fraction of J and K

Parameters
Dthe (1-particle) density matrix
Cthe occupied orbital AO coefficient matrix
scale_J_bythe coefficient of J in the result
scale_K_bythe coefficient of K in the result
target_precisionthe (absolute) target precision of the Fock matrix; this precision is not in general guaranteed, but the actual precision of the Fock matrix is usually proportional to this value.
deriv_orderthe order of geometric derivatives of the Fock matrix and energy to evaluate; the default is 0, i.e. compute standard 2-body part of Fock matrix and (if possible) energy
energy_onlyif true, will evaluate the 2-body energy (and its derivatives, if needed); the default is false
Returns
the J+K contribution to the Fock matrix and the possibly-evaluated quadratic (in D) contrubution to the energy; no non-quadratic contribution to the energy occurs, so returns only a 2-element tuple, unlike the 3-element tuple returned by FockBuild::operator()
