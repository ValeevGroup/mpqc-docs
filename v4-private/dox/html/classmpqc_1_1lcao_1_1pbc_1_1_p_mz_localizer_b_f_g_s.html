KeyVal ctor.

Parameters
kva KeyVal object; it will be queried for all keywords of zOrbitalLocalizer<Tile,Policy> as well as the following additional keywords:
Keyword Type Default Description
wfn_world class none the WavefunctionWorld
charge_exponent int 2 the charge exponent in the PM functional definition; accepted values are 2 (i.e. the standard PM functional) and 4
guess string identity how the initial guess is formed; the allowed values are identity and random
seed double 1 if guess is set to random this specifies the seed for the std::mt19937 random number generator (WARNING: standard random number generators are not portable); a seed of 0 uses the value returned by std::time() as the seed
convergence_threshold double 5e-5 the solver is converged when the norm of the gradient falls below this value
max_iter int 300 the maximum number of iterations
iters_kept int 10 the number of iterations stored in the history
bfgs_start int 10 the number of sa iterations performed before bfgs is begun
solver string bfgs specifies the numerical solver used; allowed values are bfgs, sa, cgpr, cgfr, and cghs
shift_charges string never specifies when, if ever, charges should be shifted to the ref cell; allowed values are never, always, beginning, and end
