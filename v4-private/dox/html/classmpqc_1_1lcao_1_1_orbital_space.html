Detailed Description

template<typename Tile, typename Policy, typename Derived>
class mpqc::lcao::OrbitalSpace< Tile, Policy, Derived >

an OrbitalSpace where each orbital in addition to irrep has additional attributes (e.g. occupancy, energy, etc.)

This is a CRTP base for the Derived orbital space class.

Note
The number of sets of attributes is same as or greater than the number of corresponding coefficient sets. This may occur if attributes are more specialized than coefficients, such as for spin-restricted occupied orbitals reported as a single set of orbitals but must provide the occupancies for each spincase to support not only alpha > # beta but also alpha < # beta.
