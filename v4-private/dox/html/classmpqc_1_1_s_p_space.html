Detailed Description

template<typename Basis, typename Derived>
class mpqc::SPSpace< Basis, Derived >

an SPSpace where each basis function in addition to irrep has additional attributes (e.g. occupancy, energy, etc.)

This is a CRTP base for the Derived SPSpace class.

Note
The number of sets of attributes is same as or greater than the number of corresponding bases. This may occur if attributes are more specialized than bases, such as for spin-restricted occupied orbitals reported as a single set of orbitals but must provide the occupancies for each spincase to support not only alpha > # beta but also alpha < # beta.
