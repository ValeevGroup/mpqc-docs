Detailed Description

template<typename T>
class mpqc::UCSRMatrix< T >

CSR-format matrix with unsorted rows

Note
designed to be efficient for matrices with nonzero elements in every row

this is a distributed data structure; for now every rank will contain stubs of all rows, but only data for a subset of rows
