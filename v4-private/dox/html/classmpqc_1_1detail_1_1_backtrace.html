Detailed Description

Creates a backtrace of a running program/thread. Example of use:

void make_omelet(int num_eggs) {
if (num_eggs < 1) {
mpqc::detail::Backtrace bt("breakfast fail:");
throw std::runtime_error(bt.str());
}
stove.on();
// etc.
}
