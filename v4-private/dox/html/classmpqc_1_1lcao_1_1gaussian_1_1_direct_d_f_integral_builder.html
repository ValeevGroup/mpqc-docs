Detailed Description

template<typename Tile, typename Policy>
class mpqc::lcao::gaussian::DirectDFIntegralBuilder< Tile, Policy >

DirectDFIntegralBuilder is a class that builds direct df tiles on the fly which can be composed by providing an expression. For example, the builder for the closed-shell canonical MP2 energy numerator can be computed by the following:

c++
auto direct_expr = integrals::parse_direct_expression(L"<i j|G|a b> * [ 2.0 * <i j|G|a b> - <i j|G|b a> ] = <i j a b>");
mpqc::lcao::integrals::parse_direct_expression
std::vector< std::shared_ptr< DirectExpr > > parse_direct_expression(const std::wstring &expr)
parse the direct expression
Definition direct_expressions.cpp:72

The DirectDFIntegralBuilder will build the tiles according to the left hand side of the equation. the left hand side will be parsed and provided to DirectDFIntegralBuilder as a vector of DirectExprs.

Warning
make sure there is a result type at the end of the expression e.g as above

Here's how to construct a DirectDFIntegralBuilder with expressions for closed shell MP2 energy numerator tiles:

c++
auto builder = DirectDFIntegralBuilder<Tile, Policy>(left, right,
integrals::IntegralExpr(L"(Κ|G|i a)"),
integrals::IntegralExpr(L"(Κ|G|j b)"),
integrals::parse_direct_expression(L"<i j|G|a b> * [ 2.0 * <i j|G|a b> - <i j|G|b a> ] = <i j a b>"),
integrals::ResultExpr(L"< i j a b >"));
