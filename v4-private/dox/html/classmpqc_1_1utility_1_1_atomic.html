Detailed Description

template<typename T>
class mpqc::utility::Atomic< T >

Custom overload of std::atomic to work for floating-point types (T) and std::complex<T>. Class is designed to be trivial (=bitwise) and has specialized madness::is_trivially_copyable in external/madworld/gop.h.
