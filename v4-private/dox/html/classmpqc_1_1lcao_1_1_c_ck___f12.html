Parameters
kvthe KeyVal object; all keywords from CCk class will be queried, as well as the following additional keywords
Keyword Type Default Description
approx char C approximation to compute F12 (C or D)
cabs_singles string pv1 specifies the definition of the CABS singles correction to account for the basis set incompleteness of the reference wave function. Valid values are: wp (synonymous with akw0, see DOI 10.1063/1.1562606): includes 1st-order correction for occ <-> other_uocc (CABS) mixing, neglects coupling to CC wave operator; akw (synonymous with akw1) (Adler-Knizia-Werner formulation in DOI 10.1063/1.2817618: includes 1st-order correction for occ<->complete_uocc mixing, neglects coupling to CC wave operator), pv0 (Powell-Valeev formulation, includes 1st-order correction for occ<->other_uocc mixing (other_uocc=CABS) including 1- and 2-body coupling to the CC singles), pv1 (synonymous with pv or true): includes 1st-order correction for occ<->total_uocc mixing (union of uocc and other_uocc (CABS), projecting out occ <-> uocc amplitudes from coupling to CC wave operator to avoid double-counting) including 1- and 2-body coupling to the CC singles)
vt_cabs bool true if false, omit the CABS terms in the V intermediate in VT2 and VT1 terms
