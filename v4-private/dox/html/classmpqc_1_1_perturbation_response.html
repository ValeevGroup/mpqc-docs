Parameters
kvthe KeyVal object, it will be queried for all keywords of the WavefunctionProperty class. as well as the following additional keywords:
Keyword Type Default Description
perturbation_type vector<string> {"electric","electric"} specifies the type of response to evaluate; valid entries currently include "electric"
gauge string "length" specifies the representation for the perturbation operator; valid values include "length" and "velocity"
omega vector<std::complex<real>> {0} (complex) frequency of periodic perturbation, in atomic units
Note
This constructor overrides the default target precision to 1e-6 .
