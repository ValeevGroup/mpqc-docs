Detailed Description

Molecule is a class which contains a vector of AtomBasedClusterables.

Molecule is a recursively-clustered collection of atoms.

mpqc::Molecule::Molecule ( const KeyVal &  kv)
explicit

The KeyVal constructor.

Parameters
kvthe KeyVal object. The following keywords will be queried in kv :
Keyword Type Default Description
file_name string none This gives the name of a XYZ file, from which the nuclear coordinates will be read (the XYZ format is described here). Only rank 0 in the current madness::World will read the file.
atoms array none Will query this if file_name not given. Each element of the array must specify an Atom object (see the KeyVal ctor of Atom for more info).
units string angstrom Determines the units in which the coordinates are given in atoms or in the file given by file_name. Valid choices are angstrom and bohr See UnitFactory for more information.
n_cluster int 0 If nonzero, divide the Molecule into n_cluster clusters using a k-means algorithm
attach_hydrogen boolean n_cluster==1 ? false : true if true, and n_cluster>0, cluster hydrogen atoms with the nearest non-hydrogen atom
sort_input boolean n_cluster>0 If true, sort atoms in the order of increasing distance from the center of mass; this is useful to make clustering independent of the input order of the atoms, hence the default is false if not clustering
"$:world" pointer the value returned by mpqc::get_default_world() the default madness::World object to use
Warning
this is collective if file_name is given, but is not otherwise (i.e. atoms must contain bitwise-identical contents on all ranks)

example input 1:

"molecule": {
"file_name": "water20.xyz",
"n_cluster": 20
}

example input 2:

"molecule": {
"atoms" : [
{ "element" : "O", "xyz" : [0.0, 0.0, 0.0] },
{ "element" : "H", "xyz" : [0.0, 1.0, 1.0] },
{ "element" : "H", "xyz" : [0.0, -1.0, 1.0] }
]
"n_cluster": 1
}
Note
if sort_input==true or n_cluster>1 or n_cluster==1&&attach_hydrogen==true the order of atoms in Molecule will in general differ from that provided as input. To map the atoms to the input order use Molecule::input_atom_idx()

Constructor to build Molecule from a vector of clusterables.

Parameters
[in]ca std::vector of AtomBasedClusterables used to initialize this object's Clusterables.
[in]sort_inputif true, the Clusterables are sorted based on distance from the center of mass.

Here is how to construct a Molecule from a vector of Atoms, with input coordinates given in angstroms:

c++
auto mol = std::make_shared<mpqc::Molecule>(
mpqc::Molecule(
{
{7, {0.0, 0.0, 0.55}},
{7, {0.0, 0.0, -0.55}}
}).in_units_of("angstrom").make_reclustered()
);
