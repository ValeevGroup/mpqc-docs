Detailed Description

template<typename String, SpinCase1 Spin, typename Attribute>
struct mpqc::SlaterDeterminantSparseMatrix< String, Spin, Attribute >

A (sparse) set of Slater determinants represented as a sparse matrix.

Implemented as a hashtable indexed by alpha strings, with the corresponding element containing sorted vector (by beta string hash) of determinants and their ordinal values.
