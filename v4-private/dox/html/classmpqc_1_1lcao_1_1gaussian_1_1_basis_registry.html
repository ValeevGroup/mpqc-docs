mpqc::lcao::gaussian::BasisRegistry::BasisRegistry ( const KeyVal &  kv)

Makes a standard registry of gaussian basis sets.

Parameters
kvthe KeyVal object, it will be queried for the following keywords:
Keyword Type Default Description
basis AtomicBasis none the orbital basis set
df_basis AtomicBasis none an optional density-fitting basis set
aux_basis AtomicBasis none an optional auxiliary basis set for the F12 theories
vir_basis AtomicBasis none an optional basis set for supporting unoccupied orbitals in dual-basis methods
min_basis AtomicBasis see notes a compact minimal basis; default is the Huzinaga MINI basis from GAMESS for elements up to Z=86 (Rn) except La-Lu, and ANO-RCC-MB for La-Lu and elements with 87<=Z<=96
