Detailed Description

This utility class is used to print only on node 0 and to provide attractive indentation of output.

This class only has static functions, hence there is no need to create objects of this type.

this is an old way to write singleton

◆ initialize()

void FormIO::initialize ( bool  autoindent = true)
static

Initializes the FormIO.

There should be no need to initialize FormIO since it will be auto-initialized on its first use. The only exception is when it is necessary to control the use of autoindentation (e.g. in the context of unit testing workflows do not like autoindentation).

Parameters
autoindent,iftrue calls autoindent_ostream on every std iostream (char- and wchar_t-based); set to false to turn off the autoindenting
