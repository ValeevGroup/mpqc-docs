enum class mpqc::lcao::f12::F12IntermediateShape
strong

Describes the elements of the F12 intermediate that will be computed.

Enumerator
IJIJ 

compute V^{IJ}_{IJ} for all I, J

IJJI 

compute V^{JI}_{IJ} for all I, J

IJIJ_IJJI 

compoute V^{IJ}_{IJ} and V^{JI}_{IJ} for I <= J

IJIJ_IJJI_JIIJ_JIJI 

compute V^{IJ}_{IJ} and V_{JI}_{IJ} for all I, J

This function evaluates the V intermediate, \( V_{b1 b2}^{k1 k2} \equiv r_{b1 b2}^{\alpha \beta} g_{\alpha \beta}^{k1 k2} \), (i.e., ket-Coulomb V) and the X intermediate, \( r_{b1 b2}^{k1 k2} \equiv g_{b1 b2}^{\alpha \beta} r_{\alpha \beta}^{k1 k2} \), where $g$ and $r$ are integrals over Coulomb and correlation factor (not including the cusp coefficients and spin projectors), respectively, for using in R12/F12-type theories

Parameters
lcao_factorythe LCAO factory
spacesList of indices denoting the target spaces, namely {b1, b2, k1, k2}
[in]shapethe SparseShape object used to construct the result; if default-initialized then will be computed automatically
[in]VXindicates whether to compute V ('V') or X ('X')
[in]use_cabsif false, skips the CABS-dependent terms
dfUse density fitting in integral evaluation
reduced_memoryIf df=true use direct (lazy) computation of integrals; ignored otherwise
Returns
DistArray containing \( I_{b1 b2}^{k1 k2} \), layed out as "b1,b2,k1,k2"

TiledArray::SparseShape< float > mpqc::lcao::f12::make_shape ( const TiledArray::TiledRange &  trange,
F12IntermediateShape  shape_type = F12IntermediateShape::IJIJ_IJJI 
)

computes shape of F12 intermediates in "diagonal" approximation

F12 theory contains several intermediates which in the diagonal form are sparse: only elements \( I^{ij}_{ij} \) and (possibly) \( I^{ji}_{ij} \) of the intermediate \( I^{ij}_{kl} \) must be computed. Therefore only shape elements [i,j,i,j] and (possibly) [i,j,j,i] are needed.

Note
make sure the occ is blocked by 1!
