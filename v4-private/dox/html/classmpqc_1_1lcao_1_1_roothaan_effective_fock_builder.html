Detailed Description

template<typename Tile, typename Policy>
class mpqc::lcao::RoothaanEffectiveFockBuilder< Tile, Policy >

RoothaanEffectiveFockBuilder produces spin-independent (i.e., spin-averaged) Fock matrix defining the effective spin-independent energy operator for electrons, see DOI:10.1103/RevModPhys.32.179
