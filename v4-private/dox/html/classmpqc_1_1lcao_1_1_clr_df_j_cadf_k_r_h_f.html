Detailed Description

template<typename Tile, typename Policy>
class mpqc::lcao::ClrDfJCadfKRHF< Tile, Policy >

ClrDfJCadfKRHF used density fitting with 3-center integrals stored in Clustered Low Rank (CLR) format (can be turned off optionally for J); standard density fitting is used for J and O(N) Concentric Atomic Density Fitting is used for K.
