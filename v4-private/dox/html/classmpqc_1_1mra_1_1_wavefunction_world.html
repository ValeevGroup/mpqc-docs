◆ WavefunctionWorld() [1/2]

mpqc::mra::WavefunctionWorld::WavefunctionWorld ( KeyVal const &  kv)
explicit

The KeyVal constructor.

In addition to constructing a WavefunctionWorld this changes the global madness::Function context to this world's context. If multiple such worlds exist at the same time it is necessary in each mra::Wavefunction to set the global madness::Function context appropriately.

Parameters
kvThe KeyVal object; it will be queried for all keywords queried by the KeyVal ctors of mpqc::WavefunctionWorld and mpqc::math::mra::Context, as well as the following additional keywords:
Keyword Type Default Description
units string angstrom Determines the units of lengths used to specify cell dimensions in mpqc::math::mra::Context. Valid choices are angstrom and bohr See UnitFactory for more information.
core_type string none core electron handling; the only valid value is none, i.e. treat core electrons explicitly
twobody_range string default range of 2-body interactions used to define integrals in simulation cell, valid choices are cell (particles 1 and 2 are in simulation cell) ws[R] (particle 1 is in simulation cell, particle 2 in Wigner-Seitz (WS) cell around particle 1, with WS restriction using step function smoothed to lengthscale R, in atomic units of length; R=0 corresponds to the exact/hard step function, which is not practical) and default (alias for cell for open boundary conditions, ws[1] for periodic boundary conditions).
guess_basis AtomicBasis See description The basis set to use for real-space based orbital guesses. If not provided, MINI is used.

◆ coulomb_op_inf()

const std::shared_ptr< madness::real_convolution_3d > & mpqc::mra::WavefunctionWorld::coulomb_op_inf ( ) const

accesses the operator evaluating Coulomb potential in simulation cell (i.e., convolution with the Poisson kernel) without any range restrictions imposed by twobody_kernel(). If boundary conditions are not periodic this is alias to coulomb_op().

Warning
if boundary conditions are periodic along any axis, the kernel is modified to produce absolutely convergent lattice sum for any function, including those with nonvanishing 0th moment. For functions with vanishing 0th moment this produces Ewald potential.
Returns
the shared_ptr to the Coulomb potential op
Note
(re)computes, if necessary, to match the current default MRA context

double mpqc::mra::WavefunctionWorld::nuclear_repulsion_energy ( const std::optional< Vector3d > &  electric_field = std::nullopt) const

nuclear repulsion energy attributed to the reference cell

This computes the portion of the nuclear repulsion energy involving nuclei of the simulation cell interacting via the finite-range kernel specified by twobody_kernel(). Evaluated on the first invocation and memoized.

Parameters

Note
