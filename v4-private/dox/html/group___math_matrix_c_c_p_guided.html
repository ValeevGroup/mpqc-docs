Detailed Description

Variants of canonical_column_phase() that detect degenerate panels from a given set of column attributes by trying to match to a given reference set of degenerate panel sizes.

Deducing degeneracies from orbital attributes alone is not robust, so this uses column_attributes and reference_degeneracies and adaptively deduce degenerate sets (see match_numeric_range_degeneracies()).

Dispatches to canonical_column_phase_impl().

Parameters
[in]AThe matrix to be canonicalized
[in]A_element_comparison_tolerancethe soft comparison threshold used (if needed) by compute_pd_element_rowidx() to deduce the phase-defining elements; if negative, will use the default threshold used by canonical_column_phase_impl()
[in]column_attributescolumn attributes to use for detecting the degenerate panels (via numeric_range_degeneracies())
[in]reference_degeneraciesvector of degeneracies to be matched
[in]canonicalize_panel_orderwhether to reorder degenerate panels into canonical order (see permutation_to_canonical_column_order())
[in]column_attributes_rel_eps_upper_boundmax value of the relative comparison threshold (rel_eps) to use for deducing degeneracies (see match_numeric_range_degeneracies() for the explanation); valid values satisfy rel_eps_upper_bound>0 && rel_eps_upper_bound<=2.
Returns
matrix U that canonicalizes the phase when A is right-multiplied by it (i.e. A.U)
