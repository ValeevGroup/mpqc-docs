Detailed Description

Classes/functions related to the LCAO (Linear Combination of Atomic Orbitals) numerical representation used in elecrronic structure.

Returns

Exceptions
ProgrammingErrorif
  • a and b are not supported by the same AO space but ab_ao_size_and_mappers is not provided
Note
Spin logic is punted to the user by having to specify the target space (in the future may add ability to compute the target space automatically). For now don't do anything nonsensical; what makes sense is more or less obvious; concating a spin-free with spin-aggregate space is OK, so is concating two spin-aggregate spaces, but concating two spin-specific spaces is not OK
