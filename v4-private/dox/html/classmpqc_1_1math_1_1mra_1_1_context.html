Detailed Description

defines MRA context such as boundary conditions (box), various precisions, scaling function order, etc.

Note
All precision heuristics, including eps_v_tform(), are borrowed from the SCF solver in MADNESS ... since they are not deployed consistently in MADNESS some minor tweaks are introduced

◆ eps_v()

double mpqc::math::mra::Context::eps_v ( ) const
Returns
internal precision for computing action of Coulomb potential onto a function with precision eps() to produce result accurate to eps()
Note
normally used in MADNESS for both multiplying functions and for truncating V.psi; we use this only for the latter, see eps_mul()
unless the function has many zero moments, the Coulomb potential will be long range and will not fit in typical molecule-size boxes used for open boundary conditions; this is usually not a problem since we usually care about the potential inside the box
