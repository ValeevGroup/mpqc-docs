Detailed Description

template<typename TGenerator, bool AbsHBData, typename T, std::size_t... Ranks>
struct mpqc::HBCompositeGenerator< TGenerator, AbsHBData, T, Ranks >

Combines a generator producing tuples with HBTupleGenerator.

Template Parameters
TGeneratora generator type producing generalized tuples
AbsHBDatawhether uses a HBData object containing absolute values (magnitudes) or not
Tthe values contained by the HBData object
Ranksresult rank or ranks; if Ranks is a single value, produces std::tuple<const std::array<orbital_index_type, Ranks>&, const typename TGenerator::value_type&> else produces std::tuple<std::tuple<const std::array<orbital_index_type, Ranks>&...>, const typename TGenerator::value_type&>
