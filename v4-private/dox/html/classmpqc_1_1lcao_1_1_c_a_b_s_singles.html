Detailed Description

template<typename Tile>
class mpqc::lcao::CABSSingles< Tile >

computes second-order correction due to 1-particle rotations from reference (usually occupied) to perturbing (usually CABS) space

Q_space_typespecifies the perturbing space; must be orthogonal to the reference orbitals, hence the reasonable choices are OrbitalIndex::Type::complete_uocc or OrbitalIndex::Type::other_uocc
P_space_typespecifies the reference space; the default is OrbitalIndex::Type::occ
