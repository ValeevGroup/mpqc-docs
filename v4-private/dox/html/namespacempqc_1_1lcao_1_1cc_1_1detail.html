inline

Changes the label of the given operator to the specified new label.

This function modifies the given expression containing mbpt::operator with label op_lbl by creating a new operator with label `new_label`. The new operator retains the tensor form and quantum number characteristics of the original operator.

May end up moving to SeQuant.

Parameters
exprthe expression to modify.
op_lbllabel of the operator to replace.
new_lblThe label for the new operator.
