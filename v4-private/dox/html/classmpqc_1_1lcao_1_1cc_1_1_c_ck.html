Detailed Description

template<typename Tile, typename Policy>
class mpqc::lcao::cc::CCk< Tile, Policy >

Solves for coupled cluster wave operator and properties.

This class assumes the coupled cluster ansatz of rank k (k=2 for CCSD, k=3 for CCSDT, k=4 for CCSDTQ, etc.) for the wave operator.

It can use both spinorbital and spinfree expressions. The spinorbital expressions are derived from sequant::mbpt::sr::CC class in SeQuant. These expressions can be spintraced in SeQuant to obtain expression in terms of spatial orbitals. After reading an expression, the corresponding TiledArray tensors are generated and residual equations are solved iteratively.

response:pert_lambda bool false if true, will force solve for perturbed lambda amplitudes

eom:initial_guess_size int none number of guess vectors to use for the initial guess
