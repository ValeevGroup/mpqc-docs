Detailed Description

template<std::size_t RDstF = 0, std::size_t RSrcF = 0>
struct mpqc::is_hb_replacement< RDstF, RSrcF >

Used to filter out tensor elements from HBTensor for which:

  • the first RDstF elements of dst or the remaining elements of dst are not sorted,
  • the first RSrcF elements of src or the remaining elements of src are not sorted,
  • there are coincidences within the first RDstF elements of dst or within the remaining elements of dst,
  • there are coincidences within the first RSrcF elements of src or within the remaining elements of src,
  • there are coincidences between first RDstF elements of dst and first RSrcF elements of src,
  • or, there are coincidences between the remaining elements of dst and src.
