Detailed Description

MADF generates a Density Fitting basis set (DFBS) from a user provided gaussian orbital basis set (OBS) See: https://arxiv.org/abs/2507.18471

mpqc::lcao::gaussian::MADF::MADF ( const KeyVal  kv)

The KeyVal constructor.

Parameters
kvthe KeyVal object, it will be queried for all keywords of the MADF class, and the following keywords:
Keyword Type Default Description
basis Basis The OBS basis for the molecule
min_basis Basis MINI or ANO-RCC-MB (if Z>54) The Minimal basis for the molecule
tau1 double 1.0e-6 MADF threshold for screening of DF shells with L<=2L_occ
tau2 double 1.0e-5 MADF threshold for screening of DF shells with L>2L_occ
zeta double 1.4 MADF threshold for lowest exponent ratio in each L.
kappa double 3.0 Regularization constant for kappa regularizer of t2 amplitudes
chol_tol double 1.0e-8 The threshold for the pCD screening of fused exponents.
tile_size size_t OBS size The block size of all basis sets used in MADF. Will use size of OBS set as reblock size if not provided in kv
verbose bool false Whether to print verbose output
Warning
These keywords should be prefixed by "wfn_world:" (not recommended) or defined directly in this KeyVal object
Note
Set tau1 to 1.0e-7 if the OBS is designed for spin-orbit relativistic calculations
