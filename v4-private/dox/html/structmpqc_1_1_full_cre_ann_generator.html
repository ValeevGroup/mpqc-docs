Detailed Description

template<typename String, std::size_t RC, std::size_t RA>
struct mpqc::FullCreAnnGenerator< String, RC, RA >

Generates all possible 2-tuples of {created,annihilated} index sets of ranks RC and RA from string String

Template Parameters
RCrank of tuple of created indices (can be zero)
RArank of tuple of annihilated indices (can be zero)
Stringa string type
Note
distinct from TupleOfOrbitalTuplesGenerator to be able to specialize its factory
